Votre numéro de téléphone

Nous avons besoin de votre numéro de téléphone afin que l'apiculteur souhaitant venir récupérer l'essaim puisse vous contacter facilement pour obtenir des détails. Ce numéro ne vous sera demandé qu'une seule fois jusqu'à la prochaine installation de l'application. Pour modifier votre numéro, cliquez sur l'icône de réglages dans le coin supérieur droit
